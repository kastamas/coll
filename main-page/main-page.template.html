Функционал, доступный в данный момент

для текстов

  • Добавление текста
  • Просмотр текста
  • Просмотр списа текстов
  • Редактирование текста

для словосочетаний

  • Добавление словосочетания
  • Просмотр списка словосочетаний с возможностями сортировки и фильтрации
  • Редактирование словосочетания
  • Удаление словосочетания

Будущие обновления:

В общем

  • Улучшение удобства использования приложения

Примечания:

На страницах с таблицами можно менять порядок выода, кликая по заголовкам столбцов, которые выделены синим цветом. Повторный клик по заголовку инвертирует вывод(если вывод был по возрастанию - станет по убыванию и наоборот).

После каждого действия выводится сообщение об успехе/неудаче. Оно выводится сразу после действия и отностится именно к нему, а не к предыдущим.


Рекомендации:

Советую использовать браузер Google Chrome последних версий, так как он наиболее стабильный и прогрессивный. В других браузерах тоже всё должно работать, но это не точно.
