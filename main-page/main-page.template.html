  • Отображение количества словосочетнай в тексте в списке текстов

Функционал, доступный в данный момент

для текстов

  • Добавление
  • Просмотр определённого текста
  • Просмотр списа текстов
  • Редактирование

для слвосочетаний

  • Добавление
  • Просмотр списа словосочетаний с сортировкой
  • Редактирование словосочетания
  • Удаление словосочетаний

Будущие обновления:

  • Улучшеный просмотр списка словосочетаний
  • Дополнительные фильтры для сортировки
  • Другие улучшения и прихорашивания

Примечания:

На страницах с таблицами можно менять порядок выода, кликая по заголовкам столбцов, которые выделены синим цветом.
Повторный клик по заголовку инвертирует вывод. Если вывод был по возрастанию - станет по убыванию.

После каждого действия выводится сообщение об успехе/неудаче. Оно выводится сразу после действия и отностится именно к нему, а не к предыдущим.


Рекомендации:

Советую использовать браузер Google Chrome, так как он наиболее стаблиьный. В других браузерах тоже всё должно работать, но это не точно.
